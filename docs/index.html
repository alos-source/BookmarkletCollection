Bookmarklet Directory

View on GitHub

Drag the green buttons below to your bookmarks bar. These tools run locally in your browser.

Amazon To Camel

Opens the CamelCamelCamel price tracking page for the current Amazon product.

Install Amazon To Camel

Amazon To Keepa

Opens the Keepa price tracking page for the current Amazon product.

Install Amazon To Keepa

Humble Bulk Downloader

Downloads all available Humble Bundle files in batches to avoid browser timeouts. Also provides a "Plan B" option to copy all download links for manual use in download managers like JDownloader. Only works on Humble Bundle 'Keys & Entitlements' pages, for purchased items.

Install Humble Bulk Downloader

Humble Keys

Creates a CSV export of Humble Bundle keys with title, status, key, and page link to download. Only works on Humble Bundle key redemption pages, for purchased items.

Install Humble Keys

Image Scraper

Extracts image URLs from the current webpage and exports them as a JSON file.

Install Image Scraper

Invert Site

Inverts the colors of the webpage (dark mode simulator).

Install Invert Site

Link Scraper

Extracts all links from the current webpage and exports them as a JSON file.

Install Link Scraper

Mail Share

Creates a new email with the title and link of the current page.

Install Mail Share

Qr Generator

Creates a QR code of the current URL for quick scanning with a mobile device.

Install Qr Generator

Recipe To Obsidian

Extracts recipe data from structured JSON-LD and creates a markdown note in Obsidian if installed. Asks for the Obsidian vault name on first use. Tested with chefkoch.de and kitchenstories.com, but should work with any site that uses JSON-LD.

Install Recipe To Obsidian

Rss Finder

Finds and displays RSS feeds on the current page.

Install Rss Finder

Site Shot

Captures a screenshot of the current page and downloads it as an image.

Install Site Shot

Toolbox

A toolbox bookmarklet that provides context-specific tools for various websites like recipe extraction, Humble Bundle downloads, Amazon price tracking, and YouTube channel analysis via SocialBlade.

Install Toolbox

Video Scraper

Extracts video and GIF links from the current webpage and exports them as a JSON file.

Install Video Scraper

Yt To Socialblade

Opens the Socialblade page for the YouTube channel handle of the current video or channel page.

Install Yt To Socialblade
